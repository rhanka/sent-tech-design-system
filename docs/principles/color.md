# Principes Couleur — Sent Tech

- Aucune couleur hex `#fff` / `#000` sur couleur de fond DS; préférer OKLCH via tokens.
- Les teintes de surface doivent être cohérentes avec la ligne de produit (marque + contraste a11y).
- Éviter les gradients par défaut SaaS si aucun rôle visuel fort n’existe.
- Privilégier une échelle de tokens explicite par semantic (surface, text, feedback) plutôt que des valeurs isolées.
