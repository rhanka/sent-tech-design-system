# Principes Typographie — Sent Tech

- Le titre (h1/h2) doit avoir un registre distinct du corps.
- Interdire la monoculture : 2-3 tailles de base minimum (display, body, meta).
- Réserver `line-height` dense pour les zones techniques, plus lisible ailleurs.
- Introduire des bornes de largeur de ligne (`max-width`) pour tous les paragraphes de lecture > 65 caractères.
- Préférer une hiérarchie visuelle simple et stable à la “gratification” de styles.
