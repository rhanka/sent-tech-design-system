# Principes Responsive — Sent Tech

- Prioriser une grille de rupture lisible : pas de recouvrement visuel entre sections dès 640px.
- Garder la cadence verticale constante quand la largeur chute (éviter le cramptage de spacing).
- Réserver un minimum de 44px au clavier/souris pour les cibles d’action critiques sur les tailles compactes.
- Basculer proprement les actions secondaires en colonne en mobile avec zones de clices larges et espacées.
- Prévoir des points de rupture simples, cohérents avec les composants DS (mobile/tablette/bureau).
