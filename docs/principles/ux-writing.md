# Principes UX Writing — Sent Tech

- Éviter les signes de style qui nuisent à la lecture IA-aware, en particulier `—`, `--` en microcopy.
- Préférer une syntaxe simple : verbe + objet + contexte, phrases courtes, ponctuation explicite.
- Chaque état important doit être textuellement explicite (icônes + labels là où la décision est critique).
- Supprimer les redondances de lecture (ex : URL visible quand le lien est déjà connu).
- Une ligne de texte = une unité sémantique claire : limiter les phrases multiples dans les badges/labels visuels.
